Vending Machine

Vending machine front

Vending machine backside

Vending machine internal

This is a vending machine built by Ivan, John, and myself in two and a half weeks using embedded C. It accepts quarter, dime, and nickel as payment. The LCD screen shows how much money is inserted and the selection that the user makes. It also has a Wifi unit that can track item stock. The following are things that we used to build the vending machine:

  • Microchip microcontrollers
  • ESP8266 Wifi module
  • Servo motors
  • Stepper motors
  • NMOS transistors
  • Photodiodes
  • LCD screen
  • 4x4 Keypad
